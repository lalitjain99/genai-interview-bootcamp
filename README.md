# 🎯 Why This Repository Exists

Most interview preparation resources focus on **memorizing answers**.

This repository follows a different philosophy.

The goal is to understand **why** things work before learning **how** to use them.

Every topic is taught from first principles using:

- 📖 Story-based explanations
- 🧠 Intuition before mathematics
- 📐 Mathematical foundations
- 💼 Interview-focused discussions
- 📝 Assignments to test understanding
- 🔁 Revision notes for quick recap

The objective is not just to crack interviews, but to build a deep understanding of Generative AI systems.
